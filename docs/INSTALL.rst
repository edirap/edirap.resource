edirap.resource Installation
----------------------------

To install edirap.resource using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``edirap.resource`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        edirap.resource

* Re-run buildout, e.g. with:

    $ ./bin/buildout
